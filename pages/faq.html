Frequently asked questions

Onboarding, auth & data access

Key answers for getting started with CloudDIET.

Which cloud providers does CloudDIET support?
CloudDIET currently supports Microsoft Azure.
How long does it take to get started?
It is a SaaS platform with no agents or deployment. Configure in minutes with a Service Principal; reporting starts quickly, and optimization opportunities surface over hours to days depending on environment size.
What are my authentication options?
Users sign in with Azure Entra ID / Azure Active Directory using their organizational account, and roles can be managed inside CloudDIET.
What data can CloudDIET access?
Read-only access is limited to billing, resource metrics, and Azure control-plane metadata. There is no data-plane access to customer content such as tables, schemas, queries, or files. See full security details →
Azure only No agents or deployment Azure Entra ID / SSO Read-only control-plane access
